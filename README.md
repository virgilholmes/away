# Away
Instant Messaging app for open-oscar-server
## Why
I really liked the [open-oscar-server](https://github.com/mk6i/open-oscar-server) and chatting with friends and family, but I ran into an issue. No one really cares to be on their computer a lot, if we have a computer in our pockets. Me and Codex hacked this thing together. a open-oscar-server chat app for iOS. it is in very very early development and things will change daily, but I really hope that I can get this to work some more.
